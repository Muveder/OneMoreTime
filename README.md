1- Integrantes del equipo: Cristóbal Vergara y Eric Aubel

2- Tecnología de Backend utilizada: En este proyecto se ha utilizado SpringBoot

3- Listado de Endpoints (propios y externos): 1- Método: GET, 2- Método: POST, 3- Método: PUT, 4- Método: DELETE
externos: Método: GET, URL Base: https://api.rawg.io/api/games

4- Instrucciones para levantar el servidor y la app: Primero descargar y descomprimir el archivo .zip de  backend-OneMoreTime. Luego abrirlo en IntelliJ e ir a BackendOneMoreTimeApplication.kt, darle a la flecha verde que está arriba para correr el backend, con esto descargar este git y abrirlo en android studio, esperar a que se sincronice el gradle y ejecutar la app desde la misma flecha arriba mediante un dispositivo virtual o un dispostivio real. Con esto ya estará funcionando, importante tener postman instalado en nuestro dispositivo para poder hacer las pruebas en la base de datos y los endpoints.
